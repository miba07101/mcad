Pythagorean's Theorem and Perimeter

Length of side a;

\( a = 3 \ \mathrm{} \)

Length of side b;

\( b = 4 \ \mathrm{} \)

Length of the hypotenuse

\[ \begin{align} c &= \sqrt{ {\left( {a} \right)}^{ {2} } + {\left( {b} \right)}^{ {2} } } = \sqrt{ {\left( 3 \ \mathrm{} \right)}^{ 2 \ \mathrm{} } + {\left( 4 \ \mathrm{} \right)}^{ 2 \ \mathrm{} } } \\[4pt] & \therefore c = 5 \ \mathrm{} \end{align} \]

Perimeter of the triangle

\[ \begin{align} P &= {a} + {b} + {c} = 3 \ \mathrm{} + 4 \ \mathrm{} + 5 \ \mathrm{} \\[4pt] & \therefore P = 12 \ \mathrm{} \end{align} \]
